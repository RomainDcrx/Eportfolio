1) Pour récup le projet : 
  - git clone https...
2) rentrer dans le projet : 
  - cd Eportfolio 
3) Installer les dépendances :
  - composer install
4) Avoir accès au code :
  - open folder -> Windows(C:)\Utilisateurs\Admin\Eportfolio
5) Pour sauvegarder mon nouveau code sur Github :
  - git add
  - git commit -m "message"
  - git push origin main
6) Pour lancer serveur :
  - symfony server:start
7) Pour arreter serveur :
  - symfony server:stop
